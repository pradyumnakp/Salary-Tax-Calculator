# Salary Tax Calculator (Python)

This project calculates monthly salary and income tax based on Indian tax slabs.

## Features
- Monthly salary calculation
- Dynamic salary changes
- Automatic tax recalculation
- Excel report generation using openpyxl

## Technologies Used
- Python
- openpyxl

## How to Run
1. Install openpyxl:
   pip install openpyxl

2. Run the program:
   python tax_calculator.py

## Output
- Generates Excel file: tax_report.xlsx
